# Todo.txt format for Vim

Currently only syntax highlight is supported. No extra functionality except for SortTodo command.

Unpack/clone to ``~/.vim`` or ``~/.vim/bundle`` if you have pathogen.

## License
Public domain.

## Commands
:SortTodo  
   Sort tasks by priority
